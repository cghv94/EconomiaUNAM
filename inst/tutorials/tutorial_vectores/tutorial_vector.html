1. ¿Qué es un Vector?

Introducción

Breve explicación de los Vectores

Un vector en R es la manera más sencilla de establecer. Es una colección de uno o más datos del mismo tipo y nos sirven para construir matrices, listas, data frame, etc. Lo que posibilita que se pueda hacer análisis de datos al más alto nivel.

Existen varias clases de vectores de acuerdo a su contenido, por ejemplo, los vectores con números reales se llaman Numeric, los vectores con números enteros Integer, los vectores con caracteres lógicos se llaman Logic, los vectores con texto reciben el nombre de Character y los que contienen números complejos se llaman Complex.

Con los vectores podemos realizar operaciones aritméticas que se aplican a cada uno de sus elementos. A este proceso le llamamos vectorización.

¿Cómo hacer un vector?

# Ejemplo 1 - Vector numérico
vector1 <- c(1,2,3,4,5)
vector1
# Ejemplo 2 - Vector secuencia
vector2 <- 1:10
vector2
# Ejemplo 3 - Vector de cadena de texto 
animales <- c("gato","perro","oso", "conejo")
animales
# Ejemplo 4 - Vector logico 
Verdadero <- c(TRUE, TRUE, FALSE, FALSE, TRUE)
Verdadero

Operaciones con vectores

Las operaciones aritméticas y relacionales pueden vectorizarse. Si las aplicamos a un vector, la operación se realizará para cada uno de los elementos que contiene. 

#Suma
vector_a <- c(120, 300, 921, 88)
vector_b <- c(30,100,79,2)
a_mas_b <- vector_a+vector_b
a_mas_b
#Resta
vector_a <- c(120, 300, 921, 88)
vector_b <- c(30,100,79,2)
a_menos_b <- vector_a-vector_b
a_menos_b
#Multiplicacion
vector_a <- c(120, 300, 921, 88)
vector_b <- c(30,100,79,2)
a_por_b <- vector_a*vector_b
a_por_b
#Divsión
vector_a <- c(120, 300, 921, 88)
vector_b <- c(30,100,79,2)
a_entre_b <- vector_a/vector_b
a_entre_b
